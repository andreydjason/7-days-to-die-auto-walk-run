# 7 Days to Die Auto Walk/Run
7 Days to Die Auto Walk/Run

Runs with AutoHotKey [https://www.autohotkey.com/](https://www.autohotkey.com/)

You can use the .exe directly, or the .ahk with AutoHotKey

Douple tap W to Auto Walk - hold or press Shift to run (as it is in game)

Tap W again or S to stop the Auto Walk
